# 🎹 jeux-Piano
Un petit jeu de piano interactif développé pour jouer des notes directement depuis le navigateur.

## Fonctionnalités
* Jouer des notes de piano.
* Interface simple et intuitive.
* Compatible avec les navigateurs modernes.

## Technologies
* HTML
* CSS
* JavaScript
* Installation

## Cloner le dépôt :
git clone https://github.com/NGUIMTSA-MOMO-Loveline/jeux-Piano.git
Ouvrir le dossier du projet.
Lancer le fichier index.html dans un navigateur.
